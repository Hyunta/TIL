# 노드 내장 객체 알아보기

이런게 있구나 하고 알고 있으면 된다. 외울 필요 없음

1. global : 노드의 전역 객체
   - 생략 가능하다
   - console, require, setTimeout 등등
   - global에 넣어두면 다른 파일에서도 사용할 수 있다.
   - 웬만하면 사용하지말자. 전역변수 사용하는 느낌
2. console
   - time, timeend : 시간 로깅
   - trace: 호출 스택
   - dir: 객체 로깅
3. 타이머
   - setTimeout: 몇초뒤에 실행할지 계획가능
   - setImmediate: 백그라운드에 넘겨서 코드를 동시에 실행시킬 수 있음. 비동기 코드
